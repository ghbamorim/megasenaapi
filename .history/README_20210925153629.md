# Mega api
Api para consulta dos resultados da Mega Sena

# Inicialização
Instale as dependências:
```
npm install
```
Inicie o serviço
```
npm start
```
Para executar testes
```
npx jest
```

## rotas:
## /allresults
retorna todos os resultados

## /results/:idconcurso
resultado por número do concurso

## /last
resultado do concurso mais recente
